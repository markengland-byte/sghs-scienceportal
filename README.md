# Southern Gap High School — Learning Portal

Interactive lesson modules for Biology, Anatomy & Physiology, and more.

## Structure

```
sghs-portal/
├── index.html              # Main school portal
├── biology/
│   ├── index.html          # Biology course hub
│   └── ps-cr.html          # Photosynthesis & Cellular Respiration
├── ap/
│   ├── index.html          # A&P course hub
│   └── integumentary.html  # The Integumentary System
└── bio101/                 # College Biology 101 (coming soon)
```

## Tech

- Pure HTML/CSS/JS — no build step required
- Chart.js 4.4.1 for data graphs
- Google Fonts (Playfair Display, Source Sans 3, JetBrains Mono)
- Google Apps Script for score submission

## Deployment

Connected to Vercel — push to `main` to deploy automatically.
